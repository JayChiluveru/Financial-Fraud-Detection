Explainable Financial Fraud Detection System

Developed an AI-driven fraud detection system using Machine Learning and Deep Learning models on the IEEE-CIS Fraud Detection dataset (590K+ transactions). Implemented and compared Decision Tree, Logistic Regression, Random Forest, LightGBM, XGBoost, ANN, and CNN models to handle highly imbalanced financial data.

Optimized an XGBoost model achieving 97% accuracy and 0.92 AUC-ROC, improving fraud detection performance while minimizing false positives.

Integrated Explainable AI (XAI) techniques including SHAP, LIME, PDP, Feature Importance, and SHAPASH to enhance model transparency and interpretability for regulatory and business stakeholders.

Performed feature engineering, PCA-based dimensionality reduction, SMOTE for imbalance handling, and hyperparameter tuning using Grid Search with cross-validation.

Tech Stack: Python, Scikit-learn, XGBoost, LightGBM, TensorFlow, SHAP, LIME, AWS SageMaker, Docker
